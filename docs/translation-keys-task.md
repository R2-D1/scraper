Переклад ключів іконок - потрібно пройтись по файлу translations/missing-translations/keys/part-0001.json та перекласти значення ключів на українську мову. При виконанні не використовувати ніякі автоматичні переклади, потрібно щоб ти сфокусувався сам на візуальному контексті іконок та зробив правильний переклад. Самі ключі потрібно залишати в оригінальному файлі. Для перекладу використовуй інструкцію translations/keys-translation-instruction.md. Для референсу не використовувати ніякі файли.
