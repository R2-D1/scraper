Переклад тегів для зображень Unsplash — потрібно пройтись по файлу `translations/images/missing-tag-translations/part-0001.json` (або наступному шардові файлі) та перекласти кожен тег українською. Керуйтеся правилами з `translations/images/tag-translation-instruction.md` і не використовуйте автоматичні переклади — важлива послідовність та ручний контроль якості. Записи оновлюємо безпосередньо в майстер-файлі `translations/images/tag-translations.json`, а шардовані списки залишаємо порожніми після переносу перекладів. Якщо певні службові теги не мають потрапляти у `media-meta.json`, заносьте їх до `translations/images/tag-blacklist.json` — Unsplash-скрипт пропустить такі значення і не включатиме в черги на переклад.
