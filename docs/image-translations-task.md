У файлах media-meta.json, в папках зображень library/unsplash потрібно перекласти на українську мову:
1. name зображення
2. tags
3. Оригінальні tags перенести в keys та також перекласи. Тобто в tags будуть лише перекладені tags на українську мову а в keys будуть і оригінальні tags і перекладені на українську мову + додані можливі синоніми.

Важливо - переклад потрібно робити на основі самого зображення і того що на ньому зображене. Якщо є якісь теги які не мають відношення до зображення чи являються надлишковими то прибрати їх як з тегів так і з ключів. При виконанні не використовувати автоматичні перекладачі, потрібно саме щоб ти сам це зробив на основі контексту зображення. Остаточні формулювання тегів синхронізуємо з майстер-файлом `translations/images/tag-translations.json`, а назви зберігаємо в `translations/images/name-translations.json`. Пропущені записи автоматично накопичуються у відповідних `missing-tag-translations` та `missing-name-translations`, тож після перенесення значень у майстер-файли потрібно прогнати `pnpm run images:update-translations`, щоб очистити черги.
