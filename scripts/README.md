Ця директорія призначена лише для багаторазових, загальних скриптів.

Що змінено
- Видалено тимчасові/одноразові скрипти, прив'язані до конкретних «part‑XXXX» колекцій.
- Логіка для разових прогонів має жити поза репозиторієм або зберігатися локально.

Рекомендації
- Перед додаванням нового скрипта переконайтеся, що він повторно використовуваний.
- Для експериментів або разових міграцій використовуйте `tmp/` або окремі гілки без коміту таких файлів у `main`.
